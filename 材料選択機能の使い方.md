# 材料選択機能の使い方

## 概要
建物の各パーツ（柱，床，屋根，壁，バルコニー）に対して，コンクリートまたは木材を選択できる機能です．

## パラメータ
- `material_columns`: 柱の材料（0: コンクリート，1: 木材）
- `material_floor1`: 1階床の材料（0: コンクリート，1: 木材）
- `material_floor2`: 2階床の材料（0: コンクリート，1: 木材）
- `material_roof`: 屋根の材料（0: コンクリート，1: 木材）
- `material_walls`: 外壁の材料（0: コンクリート，1: 木材）
- `material_balcony`: バルコニーの材料（0: コンクリート，1: 木材）

## 材料による違い

### 1. 構造性能（断面積の自動調整）
- **木材選択時**: 
  - 柱: 1.2倍
  - 床スラブ: 1.5倍
  - 壁: 1.5倍

### 2. コスト
- コンクリート: 20,000 円/m³
- 木材: 50,000 円/m³

### 3. 環境性能
- コンクリート: 410 kg-CO2/m³
- 木材: -836 kg-CO2/m³（炭素固定効果）

### 4. 視覚的表現
- コンクリート: グレー系
- 木材: 茶色系

## FreeCADでの色表示

FreeCADのコマンドライン版では色が自動的に適用されません．
GUIで開いた後，以下の手順で色を適用してください：

### 方法1: Pythonコンソールで実行
1. FreeCADでファイルを開く
2. メニューから「表示」→「パネル」→「Pythonコンソール」を開く
3. 以下のコードを貼り付けて実行：

```python
import FreeCAD as App
import FreeCADGui as Gui

doc = App.ActiveDocument
if doc:
    for obj in doc.Objects:
        if hasattr(obj, 'ShapeColor') and hasattr(obj, 'ViewObject') and obj.ViewObject:
            try:
                obj.ViewObject.ShapeColor = obj.ShapeColor
                obj.ViewObject.Transparency = 0
                print(f"{obj.Label}: {obj.ShapeColor}")
            except:
                pass
    print("色の適用が完了しました")
```

### 方法2: スクリプトファイルを実行
1. FreeCADでファイルを開く
2. Pythonコンソールで以下を実行：
```python
exec(open('/path/to/apply_material_colors.py').read())
```
