AI-Arch ドキュメント

🏗️ プロジェクト概要

AI-Archは、FreeCADとCalculiXを活用した建築設計最適化システムです。パラメトリックモデリング、有限要素解析（FEM）、粒子群最適化（PSO）を組み合わせ、構造的に安全で経済的、環境に優しい建築設計を自動的に探索します。

🔧 主要コンポーネント

設計変数（20個）

  • 形状パラメータ（14個）: 建物寸法、構造部材サイズ、建築的特徴
  • 材料パラメータ（6個）: 各部位の材料選択（コンクリート/木材）

評価指標（5つ）

  1. 安全性: 構造安全率（≥2.0必須）
  2. 経済性: 建設コスト（円/m²）
  3. 環境性: CO2排出量（kg-CO2/m²）
  4. 快適性: 空間品質スコア（0-10）
  5. 施工性: 施工難易度スコア（0-10）
