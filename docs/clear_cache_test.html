Chromeでキャッシュをクリアする方法：

  1. 強制リロード: Cmd + Shift + R (Mac) / Ctrl + Shift + R (Windows)
  2. デベロッパーツール経由:
    • F12でデベロッパーツールを開く
    • リロードボタンを右クリック
    • 「キャッシュの消去とハード再読み込み」を選択
  3. 設定から完全クリア:
    • Chrome設定 → プライバシーとセキュリティ
    • 閲覧履歴データの削除
    • 「キャッシュされた画像とファイル」にチェック
