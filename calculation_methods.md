# 建築評価指標の計算方法

本文書は`generate_building_fem_analyze.py`における各評価指標の計算方法を説明する．

## 1. 建設コスト（`cost_per_sqm`）

### 概要
建設コストは材料費，労務費，構造複雑性，品質グレードを考慮して計算される．

### 計算式

```python
総工事費 = 構造工事費 + 基本建築費 × 複雑性係数 + 特殊要素費
構造工事費 = (材料費 + 労務費 + 鉄筋費 + 型枠費) × 構造複雑性 × 品質グレード × 構造体積係数
```

### 主要パラメータ

#### 材料費
- **コンクリート**: 28,000円/m³
- **木材**: 87,000円/m³  
- **CLT**: 125,000円/m³

#### 基本建築費
- 150,000円/m²（仕上げ, 設備など）

#### 構造体積係数（対数的増加）
```python
structural_volume_factor = 1.0 + (
    0.3 * log(column_oversize) +
    0.2 * log(floor_oversize) +
    0.1 * log(roof_oversize) +
    0.1 * log(wall_oversize)
) * material_factor * 0.8
```

- `column_oversize`: 柱断面積 / (400×400mm²)
- `floor_oversize`: スラブ厚 / 200mm
- `material_factor`: コンクリート=1.0, 木材=1.5, CLT=0.9

#### 特殊要素による追加コスト
- 傾斜壁（>10度）: +20,000円/m²
- かまぼこ屋根（morph>0.7）: +15,000円/m²

## 2. CO2排出量（`co2_per_sqm`）

### 概要
材料製造, 運搬, 施工時のCO2排出量を計算し, 2000 kg-CO2/m²を上限とする.

### 計算式

```python
総CO2排出量 = 材料由来CO2 + 鉄筋CO2 + 運搬CO2 + 施工CO2
```

### 主要パラメータ

#### 材料別CO2排出量
- **コンクリート**: 300 kg-CO2/m³
- **木材**: -800 kg-CO2/m³（炭素固定）
- **CLT**: -600 kg-CO2/m³（炭素固定）

#### その他のCO2源
- **鉄筋**: 2.0 kg-CO2/kg × 125 kg/m³（コンクリート部分）
- **運搬**: 0.1 kg-CO2/kg-km × 50km
- **施工**: コンクリート50, 木材30 kg-CO2/m³

#### 上限制限
```python
if co2_per_sqm > 2000:
    reduction_factor = 2000 / co2_per_sqm
    total_co2_emission *= reduction_factor
```

## 3. 快適性スコア（`comfort_score`）

### 概要
居住快適性を3-8の範囲で評価する. 空間の広がり, 採光, 開放感などを考慮する.

### 計算式

```python
基本スコア = 空間の広がり × 0.35 + 採光・眺望 × 0.25 + 開放感 × 0.20 + 
            プライバシー × 0.10 + 構造的安心感 × 0.10
            
raw_score = 基本スコア × 0.8 - デザインペナルティ + バルコニーボーナス
```

### スコアマッピング（`raw_score` → `comfort_score`）
```python
if raw_score < -10:
    comfort_score = 3.0 + max(0, (raw_score + 15) / 10)
elif raw_score < -5:
    comfort_score = 3.5 + (raw_score + 10) / 5
elif raw_score < 0:
    comfort_score = 4.5 + raw_score / 10
elif raw_score < 5:
    comfort_score = 5.0 + raw_score / 5
elif raw_score < 10:
    comfort_score = 6.0 + (raw_score - 5) / 5
else:
    comfort_score = 7.0 + min(1.0, (raw_score - 10) / 5)
```

### 評価要素

#### 空間の広がり感（-10～20点）
- **天井高**: 4m以上で20点, 2.4m未満で-10点
- **スパン長**: 15m以上で20点, 8m未満で-10点

#### 採光・眺望（0～10点）
- **窓面積比**: 0.8以上で10点, 0.2未満で0点
- **階高による眺望**: 3.5m以上で10点, 2.5m未満で0点

#### ピロティによる開放感（基本-5点）
- 1階開放率>0.8: +3点
- 柱の細さ: 300mm以下で+8点, 900mm超で-8点

#### プライバシー・静粛性（基本-5点）
- 壁厚による遮音: 400mm以上で+12点, 100mm未満で-2点

#### デザインペナルティ
- 極端な傾斜壁（>30度）: +5点
- 極端な屋根形状: +0.5点

## 4. 施工性スコア（`constructability_score`）

### 概要
施工の難易度を0-10点で評価する（10点が最も施工しやすい）.

### 計算式

```python
施工性スコア = 10.0 - Σ(各種ペナルティ)
```

### ペナルティ要素
- **カンチレバー**: -2.0点
- **外部階段**: -1.5点
- **傾斜壁**: -角度/10点
- **かまぼこ屋根**: 
  - morph < 0.2: 0点
  - 0.2 ≤ morph < 0.7: -0.5点
  - morph ≥ 0.7: -1.5点
- **バルコニー**:
  - 深さ ≥ 2.0m: -1.5点
  - 深さ ≥ 1.0m: -1.0点
  - それ以外: -0.5点
- **応力集中**: 最大応力/平均応力 > 3.0で-0.5点

## 5. 安全率（`safety_factor`）

### 概要
構造安全性を材料別の許容応力と変形制限から評価する.

### 計算式

```python
応力による安全率 = 平均許容応力 / 最大応力
変形による安全率 = (建物高さ/200) / 最大変位

最終安全率 = min(応力による安全率, 変形による安全率)
```

### 主要パラメータ

#### 材料別許容応力（短期）
- **コンクリート**: 35.0 MPa
- **木材・CLT**: 6.0 MPa

#### 平均許容応力の計算
```python
平均許容応力 = 柱材料の許容応力 × 0.4 + 
              壁材料の許容応力 × 0.3 + 
              床材料の許容応力 × 0.3
```

#### 変形制限
- 基本: 層間変形角1/200以下
- 木造の場合: 上記の0.3倍（70%厳格化）
- 疲労係数: 一般木材0.6，CLT 0.7

### 木造の特別な考慮
木造建築物は変形しやすいため，以下の調整を適用：
1. 変形による安全率を0.3倍に厳格化
2. 繰返し荷重による疲労の影響を考慮（一般木材0.6，CLT 0.7）

## 各部材の体積比率

全ての計算で共通して使用される部材体積比率：
- 柱: 15%
- 床: 30%
- 屋根: 15%
- 壁: 25%
- 基礎: 10%（常にコンクリート）
- バルコニー: 5%

## 注意事項

1. **CO2排出量の上限**: 2000 kg-CO2/m²を超える場合は比例的に削減
2. **快適性スコア**: 3-8の範囲に必ずクリッピング
3. **施工性スコア**: 0-10の範囲に正規化
4. **安全率**: 応力が0の場合は無限大（inf）を返す
