# `PSO.py` 使用ガイド

## 概要
`PSO.py`は，粒子群最適化（Particle Swarm Optimization）アルゴリズムを用いて建築設計の最適化を行うプログラムです．デフォルトではコスト最小化を主目的としながら，安全率2.0以上の制約条件を満たす設計を探索しますが，目的関数は簡単にカスタマイズ可能です．

## 主な機能
- 14個の形状パラメータの最適化
- 柔軟な目的関数設定（コスト，CO2，快適性，施工性，多目的）
- 安全率制約の必須適用
- リアルタイムでの最適化進捗の可視化
- 探索履歴の詳細記録（CSV形式）
- 収束曲線と粒子分布の可視化

## アルゴリズムパラメータ

### PSO設定
```python
N_PARTICLES = 15    # 粒子数
MAX_ITER = 20       # 反復回数
W = 0.7             # 慣性重み
C1 = 1.5            # 個人の最良解(pbest)への重み
C2 = 1.5            # 群れ全体の最良解(gbest)への重み
V_MAX = 0.2         # 最大速度（探索範囲の割合）
```



## 実行方法
```bash
/Applications/FreeCAD.app/Contents/Resources/bin/freecadcmd PSO.py
```

## 目的関数

### 適応度計算
```python
def calculate_fitness(cost, safety, co2, comfort, constructability):
    """
    目的関数の計算（全ての評価指標を必須引数として受け取る）
    """
    # ===== カスタマイズ可能な部分 =====
    
    # 1. 目的関数の選択（どの指標を最適化するか）
    # デフォルト：コスト最小化
    fitness = cost
    
    # 例1）CO2排出量最小化（環境重視）
    # fitness = co2
    
    # 例2）快適性最大化（※最大化問題は負の値にする）
    # fitness = -comfort  # 快適性を最大化（負の値）
    
    # 2. 制約条件の設定
    SAFETY_THRESHOLD = 2.0      # 安全率の最小値
    PENALTY_COEFFICIENT = 100000  # ペナルティの重み
    
    # 3. 安全率制約の適用（必須）
    if safety < SAFETY_THRESHOLD:
        penalty = (SAFETY_THRESHOLD - safety) * PENALTY_COEFFICIENT
        fitness += penalty
    
    return fitness
```

- **主目的**: デフォルトは建設コスト（円/m²）の最小化
- **制約条件**: 安全率 ≥ 2.0（必須）
- **カスタマイズ**: コメントを外すことで様々な最適化目標に変更可能
  - CO2排出量最小化
  - 快適性最大化
  - 施工性最大化
  - 多目的最適化（重み付け和）

## 出力ファイル

### 1. 最適化ログ（CSV）
- **ファイル名**: `pso_optimization_log.csv`
- **内容**: 
  - 反復回数，粒子番号
  - 適応度，コスト，安全率
  - CO2排出量，快適性，施工性
  - 材料情報（固定値）

### 2. 進捗ログ（テキスト）
- **ファイル名**: `pso_progress.txt`
- **内容**:
  - 各反復でのgbest値
  - pbest平均値と標準偏差
  - 実行時間統計

### 3. 設定ファイル（CSV）
- **ファイル名**: `pso_settings.csv`
- **内容**:
  - PSOパラメータ設定
  - 設計変数の範囲
  - 固定材料設定
  - 実行開始時刻

### 4. 可視化グラフ（PNG）
- **pso_convergence.png**: 収束曲線
  - 適応度の推移
  - コストの推移
  - 安全率の推移
- **pso_pbest_safety_vs_cost.png**: 個人最良解の分布
  - 各粒子のpbestにおける安全率vsコスト
- **pso_all_positions_safety_vs_cost.png**: 全世代の粒子位置
  - 探索過程での全粒子の軌跡

## アルゴリズムの特徴

### 1. 鏡像反射境界処理
探索範囲外に出た粒子は境界で反射され，速度が反転します．これにより，境界付近の解も効率的に探索されます．

### 2. タイムアウト制御
FEM解析が20秒以内に完了しない場合，その評価はスキップされ，無限大の適応度が割り当てられます．

### 3. メモリ管理
各評価後にFreeCADのドキュメントをクリーンアップし，メモリリークを防止します．

## カスタマイズ

### 1. 粒子数と反復回数の変更
```python
N_PARTICLES = 30  # より多くの粒子で探索精度向上
MAX_ITER = 50     # より多くの反復で収束性向上
```

### 2. 材料パラメータの変更
```python
FIXED_MATERIALS = {
    "material_columns": 1,      # 木材に変更
    "material_floor1": 1,       # 木材に変更
    # ...
}
```

### 3. 目的関数の変更（詳細は`PSO_customization_guide.md`参照）
```python
# 例：CO2最小化に変更
def calculate_fitness(cost, safety, co2, comfort, constructability):
    fitness = co2  # CO2排出量を最小化
    
    # 安全率制約は維持
    if safety < 2.0:
        penalty = (2.0 - safety) * 100000
        fitness += penalty
    
    return fitness
```

### 4. 制約条件の変更
```python
# より厳しい安全率制約
SAFETY_THRESHOLD = 2.5  # 2.0から2.5に変更

# 複数の制約条件を追加
if comfort < 6.0:  # 快適性制約を追加
    fitness += (6.0 - comfort) * 10000
```

## 注意事項
1. FreeCADのコマンドライン版（freecadcmd）で実行する必要があります
2. `generate_building_fem_analyze.py`が同じディレクトリに存在する必要があります
3. CalculiXがインストールされている必要があります（FEM解析用）
4. 実行時間は粒子数と反復回数に比例して増加します（目安：15粒子×20反復で約15分）
5. 材料パラメータは固定されており，最適化対象外です

## トラブルシューティング

### ImportError
`generate_building_fem_analyze`モジュールが見つからない場合は，ファイルが同じディレクトリにあることを確認してください．

### TimeoutError頻発
- `EVALUATION_TIMEOUT`を増やす（例：30秒）
- 設計変数の範囲を狭める
- メッシュ設定を簡略化する

### メモリ不足
- 粒子数を減らす
- 反復回数を減らす
- バッチ処理で分割実行する
